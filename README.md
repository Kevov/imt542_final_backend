# IMT 542 Final Project

## Prerequisites
You need to install Node.js
You need a ```STEAM_API_KEY```, this can be requested at [this link](amcommunity.com/dev/apikey) to get info using a Steam key. You can still import an existing list through JSON without the API key.

## How to access
Go to either the front-end and back-end folder, in each folder run the following commands:
```
npm install
npm run dev
```
